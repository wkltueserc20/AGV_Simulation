AGV 模擬器 — 使用說明
========================

【怎麼開】
1. 直接雙擊「AGV_Simulator.exe」。
2. 會跳出一個黑色視窗（伺服器），請「不要關閉」它。
3. 約 2~3 秒後，瀏覽器會自動打開 http://localhost:8000
   （若沒自動開，手動用瀏覽器輸入這個網址即可）

【怎麼關】
- 關掉那個黑色視窗，就會停止。

【第一次可能出現的提示】
- Windows 可能跳出「Windows 已保護您的電腦 / SmartScreen」：
  點「其他資訊」→「仍要執行」即可（因為這個程式沒有數位簽章，屬正常）。
- 防毒軟體若攔截，請允許 / 加入信任。

【設定存檔】
- 畫面左側有「匯出設定 / 匯入設定」，可把地圖與物件存成 JSON 檔互相傳遞。
- 本資料夾內的 obstacles.json / agvs.json / map_config.json 是預設場景，
  程式會自動讀取；請勿刪除。

【注意】
- 整個資料夾要一起保留，不要只複製 exe（其他檔案是執行所需）。
- 需要 Windows 10/11 64 位元，不需要另外安裝 Python 或 Node。
